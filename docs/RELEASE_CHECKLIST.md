# v1.1 비공개 알파 체크리스트

## 저장소에서 완료된 항목

- 웹/PWA 프로덕션 빌드와 Capacitor Android 프로젝트
- 20Hz 서버 권위형 이동·사격·회피·전투·추출과 공용 하드 커버 판정
- PostgreSQL 경제 트랜잭션과 전역 결제 영수증 중복 방지
- 추출 재화 기반 장비 제작, 서버 저장형 2슬롯 로드아웃과 로컬/서버 전투 효과 일치
- 확률·천장·상품 효용 공개 보급소
- 선택 분석 기본 비활성, AI 처리 고지, 데이터 내보내기와 계정 삭제
- 키보드·마우스, 터치, 표준 게임패드와 접근성 설정
- 자동화된 클라이언트 규칙, API, 경제, 결제, 시뮬레이션, 실제 WebSocket 룸 검사
- `/health` 생존 검사와 저장소를 확인하는 `/ready` 준비 상태 검사
- 빌드·서버 버전 진단 화면과 동의 기반 오류 큐·중복 억제
- 8개 이상 동시 클라이언트 인증·입장·상태 수신 부하 스모크
- 20초 동일 세션 재접속과 준비 상태·스냅샷 간격을 측정하는 장시간 소크 테스트
- 자동 그래픽 품질, HUD 갱신 제한과 전투 이펙트 풀링
- GitHub Actions 기반 서명 AAB/디버그 APK 수동 빌드
- 알파 채널 기본 결제 숨김과 외부 피드백 URL 설정

## 외부 자격증명 또는 운영 결정이 필요한 차단 항목

- Google Play/Apple/Steam 개발자 계정과 앱 식별자 확정
- Android App Bundle/iOS 아카이브 릴리스 서명 키
- Google Play Billing, StoreKit, Steam MicroTxn 실제 어댑터와 서버 검증 자격증명
- 운영자 정보와 문의처가 채워진 공개 개인정보처리방침·이용약관 URL
- 국가별 연령 등급, 확률형 아이템·청약철회·미성년자 결제 고지
- 계정 이전·복구 수단과 고객지원 처리 절차
- 저사양 Android 및 iOS 실기기 성능·발열·메모리·네트워크 단절 QA
- 접근성, 현지화, 스토어 스크린샷·영상·설명·키워드
- 운영 모니터링, 백업 복구 훈련, 장애 공지와 보안 점검

이 항목들이 완료되기 전 v1.1은 `private alpha`이며 공개 상용 서비스로 판정하지 않습니다.
