# 게임 디자인 문서 — Vertical Slice 0.1

## 제품 판타지

플레이어는 단순히 강한 캐릭터를 소유하는 지휘관이 아니라, 멸망한 세계에서 인격을 되찾는 오퍼레이터들과 함께 생존 방식을 만들어 가는 현장 리더입니다. 긴장감은 레드 존의 손실 가능성에서, 안정감과 애착은 쉘터의 성장과 오퍼레이터 기억에서 발생합니다.

## 10분 코어 루프

1. 쉘터에서 부족한 자원과 오늘의 생성 임무를 확인한다.
2. 3명의 오퍼레이터를 편성하고 레드 존에 투입한다.
3. 환경 재해를 관리하며 적과 교전하고 자원을 수집한다.
4. 적응형 AI 디렉터가 플레이 패턴에 대응해 다음 웨이브를 바꾼다.
5. 욕심을 더 낼지, 현재 화물을 추출할지 결정한다.
6. 추출 자원으로 쉘터와 오퍼레이터를 성장시킨다.
7. 오프라인 동안 편성 외 오퍼레이터가 안전 구역 작업을 수행한다.

## 전투 설계

### 플레이어

- 직접 이동·조준·사격으로 PC 슈팅의 손맛을 유지한다.
- 피격, 환경 노출, 화물 손실 가능성이 탐사 긴장감을 만든다.
- 모바일에서는 자동 조준을 제공하되 이동·위치 선정·전술 명령의 판단은 유지한다.

### 오퍼레이터

- Vanguard: 전면 압박과 어그로 분산
- Sniper: 장거리 우선 표적 제거
- Support: 회복과 위기 대응
- Engineer: 파밍과 쉘터 생산 보조

자연어는 `DRAW_AGGRO`, `FLANK`, `HOLD`, `REGROUP`, `HEAL`, `FOCUS`, `SCAVENGE`의 안전한 명령 집합으로 변환됩니다. 캐릭터 성격은 응답과 자율 행동의 가중치를 바꾸지만 플레이어 명령을 임의로 확장하지 않습니다.

### 적응형 위협

AI 디렉터는 한 전투 세션 안에서 다음 신호를 집계합니다.

| 신호 | 대응 예시 |
|---|---|
| 높은 명중률·장시간 정지 | 스토커/우회 유닛 비중 증가 |
| 높은 이동 거리 | 추적 드론 비중 증가 |
| 과도한 누적 피해 | 잠시 압박 완화, 회복 창구 제공 |
| 높은 계정 레벨과 생존 성과 | 웨이브 크기·위협 압력 증가 |

실서비스에서는 유저별 난이도 값을 숨겨 조작감을 만들지 않고, 선택 가능한 위협 단계와 분석 로그를 제공합니다.

## 수집과 관계

| 등급 | 프로토타입 | 제품화 목표 |
|---|---|---|
| R | 기본 전술 응답, 정해진 성격 | 핵심 명령과 단기 세션 문맥 |
| SR | 역할별 전투 응답 | 전투 회고와 중기 기억 |
| SSR | 장기 기억·호감도 | 선택적 딥 토크와 개인화된 관계 서사 |

희귀도가 캐릭터의 안전 기능이나 필수 전투 명령을 제한해서는 안 됩니다. 유료 차별화는 새로운 서사, 연출, 음성, 외형, 선택적 대화 깊이에 집중합니다.

## 경제와 BM 가드레일

- 코어 모집: 명시된 확률, 20회 SSR 천장, 중복 자원 환급
- 코스메틱: 의상·보이스·테마 페르소나. 소유 캐릭터의 기존 기억을 훼손하지 않음
- 시즌 패스: 탐사 목표와 꾸미기 중심. PvP 전투력 판매 금지
- 싱크로 구독: 서버 연산량·클라우드 저장·음성 기능의 명확한 사용 한도 제공
- 금지: 호감도 상실 위협, 캐릭터의 죄책감 유발 구매 권유, 기억 삭제를 이용한 강제 과금
