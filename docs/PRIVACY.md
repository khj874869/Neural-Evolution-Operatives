# 개인정보·AI 처리 안내 초안

최종 수정: 2026-07-20

이 문서는 `Neural Evolution: Operatives` v1.1에 구현된 데이터 흐름을 설명하는 제품 초안입니다. 공개 배포 전 운영 주체명, 연락처, 사업장 소재지, 국외 이전 대상과 실제 보유 기간을 확정해 법률 검토를 받아야 합니다.

## 처리하는 데이터

- 필수: 무작위 게스트 식별자, 게임 진행도, 재화, 쉘터, 오퍼레이터, 분대 편성, 서버 전투·경제 처리 기록
- 거래 안전: 플랫폼, 거래 식별자, 상품, 구매 시각, 금액과 통화. 중복 지급과 환불 대응에 사용
- 선택 분석: 사용자가 동의한 경우에만 튜토리얼·작전·보급소 진행과 제한된 오류 종류를 기록

채팅·전술 명령 원문, 마이크 음성, 주소록, 위치, 광고 식별자는 현재 v1.1 분석 이벤트에 수집하지 않습니다. 오류 메시지는 URL과 UUID 형태 식별자를 제거하고 스택 없이 전송하며, 동일 지문은 30초 동안 중복 억제합니다.

## AI 처리

현재 전술 명령 해석과 오퍼레이터 응답은 허용된 전술 명령 집합과 규칙 기반 페르소나 엔진으로 기기에서 처리합니다. 외부 LLM·STT·TTS 제공자에게 대화 또는 음성을 전송하지 않습니다. 이후 외부 AI 기능을 추가할 경우 제공자, 전송 데이터, 보유 기간과 거부 방법을 별도로 고지하고 동의를 받습니다.

## 선택과 권리

- 첫 실행에서 선택 분석을 거부해도 모든 핵심 게임 기능을 사용할 수 있습니다.
- 설정의 `개인정보·AI 안내`에서 선택 분석을 켜거나 끌 수 있습니다.
- 같은 화면에서 서버 또는 로컬 진행 데이터를 JSON으로 내보낼 수 있습니다.
- `DELETE`를 직접 입력한 후 계정을 삭제할 수 있습니다. 진행도·분석·경제 이벤트는 삭제됩니다.
- 거래 식별자는 중복 지급 방지와 법적 의무 이행을 위해 계정과 분리해 제한적으로 보존될 수 있습니다.

## 보안과 보유

게임 서버는 TLS가 적용된 배포 환경을 전제로 하며, JWT 인증·입력 검증·중복 실행 방지·PostgreSQL 트랜잭션을 사용합니다. 보유 기간은 공개 배포 전 운영 정책과 관할 법률에 맞춰 확정해야 합니다.
